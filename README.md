# Tamagotchi 電子雞 🐣

經典 Tamagotchi 電子雞的網頁版，像素風格，純前端實現。

## 遊玩方式

直接開啟 `index.html` 或訪問：  
**https://warrenlin.github.io/Tamagotchi/**

### 操控

| 按鈕 | 鍵盤 | 功能 |
|------|------|------|
| A | A / ← | 開啟選單 / 上移 |
| B | S / Enter | 確認選擇 |
| C | D / Esc | 下移 / 關閉選單 |

點擊寵物可以摸摸牠，增加開心值。

### 功能

| 操作 | 效果 |
|------|------|
| 🍙 餵食 | 飽食度 +25 |
| ⚾ 玩耍 | 開心 +20，體力 -15 |
| 💤 睡覺 | 體力持續回復 |
| 🛁 清潔 | 清潔度 +25 |
| 💊 治療 | 治癒生病狀態 |

### 遊戲機制

- **屬性衰減**：飽食、開心、體力、清潔每 5 秒自動下降
- **生病**：任何屬性降到 0 就會生病
- **死亡**：生病 30 秒未治療
- **離線計算**：關閉網頁後重新開啟，會計算離開期間的屬性衰減
- **自動存檔**：每 10 秒存到 localStorage

## 本地開發

```bash
python3 -m http.server 8080
# 打開 http://localhost:8080
```

無需安裝任何依賴，無需建置步驟。

## 技術架構

```
├── index.html   # 蛋形機殼結構、螢幕、按鈕
├── style.css    # LCD 綠色螢幕、像素網格、火焰裝飾
├── game.js      # 16×16 像素精靈圖、遊戲邏輯、Canvas 渲染
└── CLAUDE.md    # Claude Code 開發指引
```

- **角色**：小雞（雞冠、嘴巴、雞腳），16×16 像素繪製
- **螢幕**：Game Boy 綠色 LCD（`#c8dc78`），帶掃描線效果
- **機殼**：黑色蛋形，底部紫粉火焰裝飾，銀色切角螢幕框
- **動畫**：所有動畫使用 `step-end` 呈現像素風格

## 部署

透過 GitHub Pages 從 `main` 分支自動部署。
